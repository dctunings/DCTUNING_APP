The dialog Lua Debugger (Menu Window)

 

The Lua debugger allows you to execute Lua scripts step by step and view the contents of the variables. You can reach this function via the "Re-run last LUA script" dialog (requires a configured LUA path).

 

Step/In/Out:

Keyboard: F10/F11/Shift-F11

Executes a single program step. The 3 buttons differ in their behavior with LUA sub-functions. While "Step" executes the function completely and only stops again afterwards, StepIn also allows you to step through the sub-function individually. StepOut allows you to continue executing the code until the current function is terminated.

 

Walk/Jog:

These functions work as if you keep using StepIn (F11). While Walk always pauses for 500ms, Jog works as fast as it can. Both functions are ended by a breakpoint or by the Pause button.

 

Run:

Keyboard: F5

Runs the script without debugger interaction until a breakpoint is reached or Pause is clicked.

 

Pause:

Interrupts the current script, at the next LUA command

 

Edit:

Exits the script run and starts the editor mode. With Ctrl+Space you can call up a list of possible commands for words you have started. You can save the current file with Ctrl+S. As soon as you start the script execution, the file is saved and the editor mode ends.

 

Restart:

Interrupts the current script, at the next LUA command and restarts it.

 

Close:

Closes the debugger

 

Line colors:

The yellow line is the current line. The previous lines are highlighted in gray.

 

Breakpoints:

You can set/delete breakpoints by clicking to the left of the line numbers. Breakpoints interrupt the Walk/Jog/Run functions. In editor mode, you can set breakpoints with F9.

 

Watches:

You can see global/local variables in the windows at the bottom left/right. You can also query variables using the mouse tooltip and view them in the virtual comments that appear n the line during execution.

 

Notes:

The editor allows a maximum of 2048 characters per line. This is a third-party limitation that we cannot currently change.
