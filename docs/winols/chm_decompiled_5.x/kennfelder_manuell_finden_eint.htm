Manually find maps (Typing)

You can also create maps by typing the data manually. Simply press "K" (or "Selection > Selection->Map") anywhere in the hexdump. This will create a map at this address and open the dialog "Properties: Window". Now you can enter parameters like name or size.

 

Tip: The map window in the background is automatically updated to give you feedback.
