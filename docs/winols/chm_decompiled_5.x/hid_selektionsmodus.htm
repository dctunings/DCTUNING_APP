The command Selection mode (Menu Selection)

This command offers a choice of 4 different selection modes:

1) Selection of a consecutive area

2) Selection of a rectangular area

3) Selection of columns

4) Selection of rows

 

Tip:

If you just created a selection and then use this command to change the mode, then this change will also be applied to the created selection.

 

Shortcuts

Symbol bar:
Keyboard:Ctrl+Shift+M (only consecutive/rectangular)

 

Note:

If the hotkey doesn't work, Radeon graphics card driver tool is reserving it. Test it on F12>Debug.
