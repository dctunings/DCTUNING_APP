International characters
Expand All   Collapse All

WinOLS supports various languages. You can select the current language under F12 > Language.

 

1.Wrong characters in menus and dialogs:
Please check in Windows if the "Language for Unicode incompatible programs" is set correctly:
Windows Settings > Language and Region > Administrative Language Options > Administration

 
2.Wrong characters in hexdump and map list:
Please check if the "Script" of the font is correctly selected in WinOLS under:
F12 > View > Select font

 
3.Ini-Files and Combo-Box-Templates:
Save the file in "Ansi" or "UTF-8 with BOM" format.
