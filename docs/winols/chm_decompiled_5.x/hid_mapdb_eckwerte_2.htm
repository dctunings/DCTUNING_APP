Map database - Basic values - Create

This function requires WinOLS 5.00

 

Important:

This is a tool for advanced users, not an automatic wizard.

It is not enough to click on "Create". You have to understand and check each rule.

 

The create links can create a rule that matches:

- the current map

- all maps in the preview window

- all selected maps in the preview window

 

In general:

The more maps you select, the more unspecific (=worse) the resulting rule will be.

 

What to do:

·Try finding maps that are very similar. Not only by name, but also by size + content. Selected them and use the create from selection link.
·If you have different clusters of maps (with the same name, description, etc.), use one entry per cluster in the condition list.
(Cluster means that some are very similar; And others are also very similar to themselves, but different from the first cluster)
·Test the rule: Use the preview dialog to test if your rule matches maps, that it shouldn't match.
