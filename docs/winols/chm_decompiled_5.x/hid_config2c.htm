The dialog Configuration / Miscellaneous / Debug (Menu Miscellaneous)

 

LeakDebugger

Activate the LeakDebugger if WinOLS becomes increasingly sluggish during longer periods of work. Observe which values in the WinOLS title bar increase continuously. (Fluctuations and one-time increases are normal for some program functions).

Dont load all...

Under 95/98/ME/NT the number of DLLs that can be loaded at the same time is limited. If you have licensed many plug-ins, it may happen that some of these plug-ins do not work. Normally this option is automatically activated in this case, but you may also activate it manually. You can still use all plug-ins, but they may be slightly slower.
