Map database - Conditions

This function requires WinOLS 5.00

 

A rule of the map database contains one or more conditions. These can be found in this list. If you click on an entry, the details of the condition appear to the right of it in the Basic Values or Search Text area.

 

Groups:

Conditions can be arranged in groups. In total, at least one rule from each group must apply. Within a group there is an "or" condition, the groups are linked by an "and" condition. So you can determine (by 2 rules in 1 group) that a map must be exactly 8x12 or 10x14.

 

Preview:

This area shows live if a condition is true. Normally the current map window is used for this. But if you click in the list in the preview area, the clicked map is used.
