WinOLS as service

You can run WinOLS as service to perform automated tasks in the background. This option is only applicable for advanced Windows users.

 

Requirements:

·A windows PC with WinOLS license
·OLS530 (LUA for WinOLS)
·Experience in handling services

 

Steps for the installation:

1.Install WinOLS normally, Register + install plugins, check if everything is registered
2.Change HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Windows\NoInteractiveServices to 0
3.Start the service "Interactive Services Detection" (Erkennung interaktiver Dienste)
4.Open a Windows commandline with Admin rights and run these commands:
sc delete WinOLS
(Deletes an old service, if it exists)
sc create WinOLS start= auto type= interact type= share DisplayName= "WinOLS running as service" binpath= "\"C:\Program Files (x86)\EVC\WinOLS\ols.exe\" /Service \"X:\<YourPath>\<YourSkriptfilename>.lua\""
5.Edit the WinOLS service and, if desired, change the "Recovery" options to "Restart"
6.Start the WinOLS service
7.In the taskbar click on "Interactive Services Detection" (Erkennung interaktiver Dienste) and display the message
8.Register WinOLS under this account, too.
9.Close the message
10.Stop the WinOLS service and restart

 

Note:

After the successful start of the service it can still take time until LUA is started.
