Projects

All eprom data will be administered in so-called projects by WinOLS, which consist of an original file and as many modified ones (versions) as are needed. They will be saved to one single file together with the vehicle data as well as possible comments.

 

Please note:

It is not possible to edit external file formats directly. Since WinOLS stores lots of additional information you have to import it into a WinOLS project first (for example by drag+drop) and export (or burn or write) it later again.
