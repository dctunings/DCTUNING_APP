Shadow data

Some of the export plugins require additional data (beyond the hexdump data) for the export. This is the "shadow data", as it is are usually barely noticed. WinOLS automatically saves it in the project file. (per project, not per version.)

 

If a project contains shadow data, you can see this in the project properties (in the project size field).

 

They are also relevant for similar projects. WinOLS warns you if you choose an action which causes the loss of the shadow data.

 

As a consequence, these plugins cannot export without this data, so this option may not be offered at all. You can only export the projects you have imported with the same plugin.
