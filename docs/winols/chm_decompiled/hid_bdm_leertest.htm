The dialog BDM / Testempty (Menu Hardware)

 

Allows to test you if the BDM/ECU is empty and to delete it.

 

Elements are printed bold, if they contain contain differences between original and version (in the PC memory  not differences between ECU and PC).

 

Note:

BDM operations are only supported if the project type was set to "BdmToGo" in the Project properties or if the project is empty.
