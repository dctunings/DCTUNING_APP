Additional file formats

WinOLS can already import / export several file formats. You can extend this list with 2 methods:

 

a) Import / Export Plugins

Visit www.evc.de/en/product/ols/plugins.asp to find import/export plugins for several special file formats. You can buy / install them like checksums, then they will be available in WinOLS just like other file formats.

 

b) External Converters [Requires programming knowledge]

If you use other / your own file formats, you can integrate them into WinOLS. This will cause WinOLS to show then in the file dialogs. If the user selects such a file format, then WinOLS will import / export an uncompressed EcuX file and run a program provided by you to do the actual conversion from/to EcuX.
