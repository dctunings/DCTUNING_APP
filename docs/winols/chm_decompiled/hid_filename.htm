The dialog Enter a filename

 

This dialog asks you to enter a filename for a new project. Most of the time WinOLS can automatically create a new filename. This dialog appears only if youve turned the automatic naming off or if the given namespace is full (meaning all files that can be created with the namespace already exist).

 

Both issues can be configured in the dialog Miscellaneous > Configuration > Automatically. For more information about namespaces, please refer to the help page of that dialog.

 

If you are unsure what to enter in this dialog either accept the default value or just enter a large random number.
