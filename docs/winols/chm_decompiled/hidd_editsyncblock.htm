The dialog Sync block Properties

 

This dialog allows you to add or modify a sync block. A sync block is a method to make sure that a memory ranges which are currently identical will stay identical. If a modification is done in one memory range, the same modification will be performed in the other range, too.

 

You can reach this dialog, by clicking the small black triangle next to "Add" in the checksum dialog.

 

Note about addresses: The addresses in this dialog do not refer to the current element, but to the addresses like they are visible in the view <All elements>. This makes actions possible which apply to the data of multiple elements at once.
